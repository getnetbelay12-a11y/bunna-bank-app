# Staff and Manager Flow

## Loan officer flow
Login to web
-> dashboard
-> see assigned loans
-> open application
-> review details/documents
-> add notes
-> request more info / reject / approve / escalate
-> system logs activity
-> system sends notifications

## Branch manager flow
Login to web
-> branch dashboard
-> see branch totals
-> see pending approvals
-> see staff performance today/week/month/year
-> review bottlenecks
-> export report

## District manager flow
Login to web
-> district dashboard
-> compare branches
-> review district-level approvals
-> see overdue cases
-> see top/low-performing branches and staff

## Head office flow
Login to web
-> enterprise dashboard
-> review large loans
-> monitor approval pipeline
-> monitor district performance
-> monitor aggregate transaction volume
-> create and manage shareholder vote events
-> monitor voting participation and live results

## Governance admin flow
Login to web
-> voting dashboard
-> create vote event
-> enter title, description, type, dates
-> add candidates/options
-> publish vote
-> system notifies all shareholder members
-> monitor total participation
-> monitor branch-level participation
-> close vote
-> publish results
-> export audit and result summaries

## Staff performance tracking
Track by banker:
- customers helped
- transactions completed
- loans created
- loans reviewed
- loans approved
- school payments processed
- total transaction amount

## Voting dashboard metrics
- total shareholders
- votes cast
- participation rate
- participation by branch
- participation by district
- live result percentages by option
- vote status: draft/open/closed/published

## Time filters
- today
- this week
- this month
- this year
- custom range
