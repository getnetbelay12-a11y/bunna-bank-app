# Account Creation

## Flow

1. Enter phone number
2. Check existing account
3. Verify OTP
4. Enter personal details
5. Select region, city, preferred branch
6. Upload Fayda front and back
7. Submit selfie reference for identity confirmation
8. Set PIN
9. Submit onboarding request

## Required Fields

- first name
- last name
- phone number
- region
- city
- preferred branch
- Fayda front
- Fayda back
- selfie reference
- PIN

## Verification Notes

- Branch names are shown to customers, not internal branch IDs.
- Official Fayda validity must not be claimed unless integrated with an approved service.
- When official verification is unavailable, use pending or manual review states.
