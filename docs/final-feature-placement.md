# Final Feature Placement

## Home Structure Preserved

The Home page keeps the familiar CBE Bank structure:

- welcome header
- balance card
- shortcut icon grid
- banner/service area
- bottom navigation

Home was simplified instead of redesigned. Extra dashboard-style summary tiles were removed.

## Feature Placement

- `Home`
  - urgent reminder strip only when relevant
  - small service/banner area
- `Transactions`
  - transaction history
  - mini statement access
  - loan payment reminders
  - insurance renewal reminders
- `My Bank`
  - Auto Payment
  - ATM Order
  - Loan Tracker
  - Add Member On Account
  - Shareholder Voting
  - Shareholder Messages
  - Branch / ATM Locator placeholder
  - Service Utilities placeholder
- `Profile`
  - Security
  - Account Lock
  - Update Phone Number
  - KYC Verification
  - Live Chat / Support
  - Beneficiary Management
  - Terms and Conditions
  - Logout

## Selfie Requirement

Selfie is required or prepared for:

- account creation
- identity confirmation during onboarding
- phone number update when policy requires it
- add member on account

## Light Blue Usage

Light blue is used only for presentation/demo emphasis on new features:

- `New` badges
- new-feature info cards
- reminder strips
- feature highlight borders

The main CBE Bank branding remains dark blue, yellow, and white.
